# CURRENT_DATABASE

Returns the current database name

## Syntax
{: .no_toc}

```sql
CURRENT_DATABASE()
```

## Return Types
`TEXT`

## Example
{: .no_toc}

```sql
SELECT current_database()
```

**Returns**: `account_db`
