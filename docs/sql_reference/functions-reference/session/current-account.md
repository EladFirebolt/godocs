# CURRENT_ACCOUNT

Returns the current account name

## Syntax
{: .no_toc}

```sql
CURRENT_ACCOUNT()
```

## Return Types
`TEXT`

## Example
{: .no_toc}

```sql
SELECT current_account()
```

**Returns**: `test_account`
