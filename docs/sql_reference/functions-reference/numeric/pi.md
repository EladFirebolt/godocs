# PI

Calculates π as a `REAL` value.

## Syntax
{: .no_toc}

```sql
PI() 
```

## Return Types 
`DOUBLE PRECISION` 

## Example
{: .no_toc}

This example returns π as a `DOUBLE PRECISION`: 
```
SELECT
    PI();
```

**Returns**: `3.141592653589793`
