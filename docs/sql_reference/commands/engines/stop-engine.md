# STOP ENGINE

The `STOP ENGINE` statement enables you to stop a running engine.

## Syntax

```sql
STOP ENGINE <engine_name>
```

| Parameter       | Description                          | Mandatory? Y/N |
| :--------------- | :------------------------------------ | :-------------- |
| `<engine_name>` | The name of the engine to be stopped | Y              |
