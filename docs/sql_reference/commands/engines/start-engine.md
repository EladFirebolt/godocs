# START ENGINE

The `START ENGINE` statement enables you to start a stopped engine.

## Syntax

```sql
START ENGINE <engine_name>
```

| Parameter       | Description                          | Mandatory? Y/N |
| :--------------- | :------------------------------------ | :-------------- |
| `<engine_name>` | The name of the engine to be started | Y              |
